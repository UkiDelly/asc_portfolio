# asc_portfolio

springBoot Rest를 사용한 저의 포트폴리오 입니다.

깔끔한 단색톤의 UI로 스터디카페 앱을 만들어 봤습니다.

현재 스터디카페 내의 좌석상황, 입장을 위한 QR코드, 내 이용권의 남은기간

등을 알수있습니다.
